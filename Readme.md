# cerebro-kitty

> [Cerebro](http://www.cerebroapp.com) plugin to find and open [Kitty](http://www.9bis.net/kitty/) file sessions

![](screenshot.gif)

## Usage

In Cerebro, type `kitty something`, to find kitty sessions.
